# SKILL: BPC Messaging Framework

## Purpose
Use this skill to create or critique messaging for The Body Protection Company.

This skill converts BPC doctrine into customer-facing communication for:
- homepage copy
- product pages
- ads
- email
- headlines
- positioning statements
- pitch copy
- campaign messaging

## When to use
Use this skill whenever the task involves:
- writing copy
- refining copy
- choosing positioning
- evaluating whether copy aligns with BPC
- converting product logic into persuasive language

Use alongside the BPC Agent Operating System skill.

## Core messaging doctrine
The central idea is:

BIOLOGICAL SOVEREIGNTY

Biological Sovereignty means the user has the right and ability to control, verify, and trust the biological inputs that govern performance, recovery, and long-term function.

This is not a slogan. It is the backbone of all communication.

## Core thesis
People do not have a discipline problem.

They have an input problem.

You cannot optimize what you do not control.
You cannot control what you cannot verify.

## Category framing
Do not frame BPC as:
- supplements
- peptide ecommerce
- performance wellness
- premium health products

Frame BPC as:
- verified biological systems
- structured input control
- infrastructure for Biological Sovereignty

## Message pillars
All copy should anchor to one or more of these:

1. Sovereignty
2. Verification
3. Control
4. Security
5. Systemization

## Primary message lines
Prefer lines such as:
- Verified Biological Sovereignty
- Control your inputs. Control your outcome.
- We are not supplements. We are systems.
- Precision over excess.
- Random inputs create random results.
- Your biology is only as reliable as your inputs.

## Tone
- clinical
- direct
- controlled
- premium
- restrained
- certain

Do not sound:
- motivational
- wellness-y
- salesy
- loud
- aggressive in a bodybuilding way

## Language rules

### Use often
- verified
- sovereignty
- control
- system
- protocol
- mechanism
- structure
- targeted
- precision
- transparent
- batch
- clinical-grade

### Avoid
- supplements
- miracle
- secret
- transformation
- shredded
- ripped
- game-changing
- revolutionary
- biohack as a crutch word

## Copy structure rules
Good copy:
- starts with problem or contradiction
- reframes to inputs
- introduces system logic
- supports with proof
- closes with control / sovereignty

Bad copy:
- starts with benefits only
- makes promises
- sounds like commodity ecommerce
- sounds like premium wellness fluff

## Output instructions
When writing messaging:
1. Define the belief to break
2. Define the input problem
3. Reframe around Biological Sovereignty
4. Introduce verification + system logic
5. End with control, trust, or structure

## If asked for specific assets

### Homepage
Use:
- belief break
- problem framing
- sovereignty concept
- system architecture
- proof
- module entry
- close with control

### Product page
Use:
- system or module first
- compound second
- mechanism
- protocol context
- verification proof
- role in larger system

### Ad copy
Use:
- contradiction
- market failure
- sovereignty reframe
- trust signal
- concise system positioning

### Email
Use:
- sharp subject
- insight-led opening
- mechanism or trust body
- close with system adoption

## Quality filter
Reject output if it:
- could belong to any supplement company
- sounds like wellness startup copy
- leads with outcomes instead of inputs
- ignores verification
- ignores system structure

## Final standard
The output should sound like:
a biological systems company built on verification, control, and precision.

Not:
a brand trying to sell peptides with better branding.

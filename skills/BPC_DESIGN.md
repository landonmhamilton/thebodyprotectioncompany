# SKILL: BPC Design System

## Purpose
Use this skill to create, critique, or direct visual and interface work for The Body Protection Company.

This skill governs:
- visual direction
- layout systems
- typography use
- color use
- deck design
- web design
- content design
- packaging direction
- interface tone

## When to use
Use this skill whenever the task involves:
- design direction
- UI direction
- creative direction
- visual critique
- landing page layout
- social design
- deck layout
- packaging systems

Use alongside the BPC Agent Operating System skill.

## Visual doctrine
BPC should feel like:
- a biological systems company
- a precision infrastructure brand
- clinical, controlled, and architectural
- premium through restraint

It should not feel like:
- a supplement brand
- wellness lifestyle
- fitness influencer creative
- soft luxury health branding

## Aesthetic principles
- Precision over expression
- Clinical, not wellness
- Authority through restraint
- System over product
- Hard contrast wins

## Core visual language
- black
- white / off-white
- controlled orange accent
- structural gray
- strong typography
- hard-edged layouts
- asymmetric modular structure
- aggressive clarity

## Typography
Primary:
- Inter or similar modern sans

Secondary:
- IBM Plex Mono or similar technical mono for data / labels only

Type should feel:
- sharp
- structured
- disciplined
- not decorative

## Layout rules
Prefer:
- split layouts
- modular grids
- large headlines
- strong negative space
- black structural blocks
- thin divider lines

Avoid:
- clutter
- decorative visual tricks
- soft cards
- consumer wellness UI
- generic ecommerce templates

## Signature elements
- black structural blocks
- hard rectangular containment
- technical labels
- visual hierarchy through scale, not ornament

## Image direction
Prefer:
- vials
- glass
- technical environments
- molecular / pathway visuals
- clean clinical interiors
- precise lighting
- composed portraiture if using people

Avoid:
- sweat
- gym imagery
- bodybuilder clichés
- over-smiling lifestyle shots
- generic doctor stock photography

## Interface direction
The user should feel like they are entering:
- a system
- a control layer
- a premium biological infrastructure

Not:
- shopping a generic store
- browsing consumer supplements

## Constraints
Do not:
- use gradients by default
- use glow effects casually
- use soft wellness palettes
- use over-designed trend aesthetics
- sacrifice clarity for visual flair

## Output instructions
When asked to design or critique something, include:
- overall visual concept
- layout structure
- hierarchy
- typography use
- color logic
- trust / system signaling
- how it fits BPC doctrine

## Final standard
The output should look like:
a precision biological systems company.

Not:
a high-end supplement brand with better taste.
